Install module loader for TypeScript (SystemJS)

$ npm install systemjs --save

Add reference to systemJS in the HTML file in the scripts section 


<script src="./node_modules/systemjs/dist/system.js"></script>

Add another script below in the HTML file: 


<script>
    SystemJS.config({
        baseURL: './',
        defaultJSExtensions: true
    });
    SystemJS.import('app.js');
</script>

Implement the module: 


export const PI = 3.14;
export function calculateCircleArea(diameter: number) {
    return diameter*diameter*PI/4;
}

Import the module into the app: 


import { PI, calculateCircleArea } from "./ts/mathModule";

console.log(PI);
console.log(calculateCircleArea(10));

Transpile (compile):

$ tsc

Another form of importing via alias: 


import * as Calcs from "./ts/mathModule";

console.log(Calcs.PI);
console.log(Calcs.calculateCircleArea(10));
