Lite Server is a NPM plugin that is continuously running an HTML file. It's very convenient when working on a HTML/CSS code. To take advantage of lite-server do the following:
  • Make sure npm is set up:
    $ npm init
  • Install lite-server:
    $ npm install lite-server --save-dev
  • In package.json file, in the script section, tell npm to start lite-server: 
    
                ...
                "scripts": {
                "test": "echo \"Error: no test specified\" && exit 1",
                "start" : "lite-server"
                },
                ...
  • Run lite-server by running:
    $ npm start
    This will run lite-server in the background using localhost, and every time index.html is changed it will be automatically refreshed.
