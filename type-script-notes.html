1. TypeScript compiles to JS
  2. Web Site: www.typescriptlang.org
  3. Installation:
    npm install typescript -g
  4. Create a .ts file, for example: sample.ts
  5. Compile:
    tsc sample.ts
    This will create a file: sample.js
  6. To automate the process, type:
    $ tsc --init
    This will create tsconfig.json file, which configures the way tsc runs. This allows to run just:
    $ tsc
