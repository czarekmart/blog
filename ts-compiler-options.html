compilerOption Section

  • noEmitOnError - do not produce js output if there are errors. Default = false
  • sourceMap - creates a map file which allows to debug ts file in the browser. Default = false
  • noImplicitAny - if true then all declarations must have a type (or "any" for traditional JavaScript). Default = false.
  • noUnusedParameters = reports error on unused parameteres. Default = false.
  • strictNullChecks - error if assigning null to non-nullable types (numbers). Default = false.
